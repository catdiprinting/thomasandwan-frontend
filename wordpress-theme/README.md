# Thomas & Wan WordPress Theme

A professional WordPress theme for Thomas & Wan, a women-owned medical malpractice law firm in Houston, Texas.

## Features

- **Custom Post Types**: Testimonials, Practice Areas, Case Results, Team Members
- **One-Click Demo Import**: Import all content with a single click
- **Customizer Options**: Easily update contact info, social media, and homepage content
- **Responsive Design**: Looks great on all devices
- **SEO Optimized**: Clean, semantic markup
- **Fast Loading**: Minimal dependencies, optimized CSS

## Installation

1. Download the `thomas-wan-theme` folder
2. Zip the folder if not already zipped
3. In WordPress Admin, go to **Appearance > Themes > Add New > Upload Theme**
4. Upload the zip file and click **Install Now**
5. Click **Activate**

## Demo Import

After activating the theme:

1. Go to **Appearance > Import Demo**
2. Click **Import Demo Content**
3. Wait for the import to complete

This will create:
- 6 Testimonials
- 5 Practice Areas with full content
- 3 Case Results
- 2 Team Members (Linda Thomas & Michelle Wan)
- Core pages (Home, About, Contact, Testimonials, Blog)
- Navigation menus

## Theme Customization

### Contact Information
Go to **Appearance > Customize > Contact Information** to update:
- Phone number
- Address
- Email

### Social Media
Go to **Appearance > Customize > Social Media** to add:
- Facebook URL
- Twitter URL
- LinkedIn URL
- Instagram URL

### Homepage Settings
Go to **Appearance > Customize > Homepage Settings** to update:
- Hero headline
- Hero subheadline

### Custom Logo
Go to **Appearance > Customize > Site Identity** to upload your logo.

## Content Management

### Adding Testimonials
1. Go to **Testimonials > Add New**
2. Enter a title (for internal use)
3. Write the testimonial in the content area
4. Fill in the sidebar fields:
   - Client Name (displayed on frontend)
   - Date (optional)
5. Publish

### Adding Practice Areas
1. Go to **Practice Areas > Add New**
2. Enter the practice area title (e.g., "Birth Injuries")
3. Add excerpt (short description for cards)
4. Add full content
5. Fill in sidebar fields:
   - Icon (emoji or dashicon)
   - Cases We Handle (one per line)
6. Set featured image
7. Set menu order for display order
8. Publish

### Adding Case Results
1. Go to **Case Results > Add New**
2. Enter a title
3. Add description in content
4. Fill in sidebar fields:
   - Settlement/Verdict Amount
   - Case Type
5. Publish

### Adding Team Members
1. Go to **Team Members > Add New**
2. Enter the person's name as title
3. Add bio in content
4. Fill in Job Title in sidebar
5. Set featured image (portrait photo)
6. Set menu order
7. Publish
