## How to Use

### Step 1: Create a New Page in WordPress
1. Go to **Pages > Add New**
2. Enter the page title (e.g., "About Us")

### Step 2: Switch to Code Editor
1. Click the **three dots** (â‹®) in the top-right corner
2. Select **Code editor** (not Visual)

### Step 3: Copy and Paste
1. Open the corresponding HTML file
2. Copy ALL the code (Ctrl+A, Ctrl+C)
3. Paste into WordPress Code Editor (Ctrl+V)

### Step 4: Switch Back to Visual
1. Click **Exit code editor** at the top
2. You should see the formatted page with all sections
3. Click **Publish** or **Update**
